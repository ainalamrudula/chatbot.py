# 🤖 Simple Rule-Based Chatbot  ## 📌 Project Overview  This project is a **Simple Rule-Based Chatbot** developed using Python. The chatbot interacts with users through the console by using **pattern matching and predefined rules** to generate responses.  It is designed as part of an Artificial Intelligence project to demonstrate the basics of conversational systems.  ---  ## 🚀 Features  * 💬 Responds to user input using rule-based logic * 🔍 Uses pattern matching with regular expressions * 📚 Includes a small knowledge base (AI, Python, Chatbot) * 🧠 Supports basic intents:    * Greetings   * Help   * Small talk * 📝 Logs conversation history into a file (`chat_log.txt`) * 🖥️ Runs interactively in the console  ---  ## 🛠️ Technologies Used  * Python * Regular Expressions (`re` module) * File Handling  ---  ## ▶️ How to Run  1. Clone the repository:     ```bash    git clone <your-repo-link>    ```  2. Navigate to the project folder:     ```bash    cd chatbot-project    ```  3. Run the program:     ```bash    python chatbot.py    ```  ---  ## 💡 Example Interaction  ``` You: hello Bot: Hello! How can I help you today?
